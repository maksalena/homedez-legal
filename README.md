# Политика конфиденциальности Homedez

Дата вступления в силу: 14 февраля 2026 г.

## Какие данные мы собираем

При использовании приложения Homedez мы собираем следующие данные:

- **Имя** — для идентификации в личном кабинете
- **Адрес электронной почты** — для регистрации, входа и связи
- **Номер телефона** — для связи по заказу
- **Адрес объекта** — для выполнения услуги
- **Фотографии** — при добавлении к заказу (по вашему желанию)
- **Комментарии и оценки** — при оформлении заказа и отзыва

## Как мы используем данные

Собранные данные используются исключительно для:
- Оформления и выполнения заказов на обработку от вредителей
- Связи с вами по вопросам заказа
- Улучшения качества обслуживания

## Хранение данных

Данные хранятся на защищённых серверах Supabase (облачная платформа). Передача данных осуществляется по зашифрованному протоколу HTTPS.

## Передача третьим лицам

Мы не продаём и не передаём ваши данные третьим лицам, за исключением случаев, предусмотренных законодательством РФ.

## Отслеживание

Приложение не использует рекламные трекеры и не отслеживает действия пользователей в маркетинговых целях.
